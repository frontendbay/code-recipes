import { Steps } from '@astrojs/starlight/components';

<Steps>

1. In your project's source directory, navigate to the styles folder and open the global css file and set up your base Tailwind CSS configuration:
    ```css {8-11}
    @tailwind base;
    @tailwind components;
    @tailwind utilities;

    /* Your base theme settings */
    @layer base {
      html {
        --color-primary: #4285f4;
        --color-secondary: #34a853;
        --color-buttons: #fbbc05;
        --color-typography: #ea4335;
      }
    }
    ```
2. Create individual CSS files for each theme variant. Example `uno.css`, `dos.css` etc.
    ```css
    /* src/styles/themes/theme-uno.css */
    html[data-theme="uno"] {
      --color-primary: #f98866;
      --color-secondary: #80bd9e;
      --color-buttons: #89da59;
      --color-typography: #ff320e;
    }
    ```

    ```css
    /* src/styles/themes/theme-dos.css */
    html[data-theme="dos"] {
      --color-primary: #f4cc70;
      --color-secondary: #6ab187;
      --color-buttons: #de7a22;
      --color-typography: #20948b;
    }
    ```

3. In `global.css` Below your base theme settings, you can start importing your theme variants.
    ```css {10-11}
    /* Your base theme settings */
    @layer base {
      html {
        --color-primary: #4285f4;
        --color-secondary: #34a853;
        --color-buttons: #fbbc05;
        --color-typography: #ea4335;
      }

      @import "themes/theme-uno.css";
      @import "themes/theme-dos.css";
    }
    ```

4. Open the `tailwind.config.js` file and configure it to extend the theme colors using the CSS custom properties defined in your global.css
    ```ts {7-12}
    /** @type {import('tailwindcss').Config} */

    module.exports = {
      content: ["./src/**/*.{html,js,jsx,ts,tsx}"],
      theme: {
        extend: {
          colors: {
            primary: 'var(--color-primary)',
            secondary: 'var(--color-secondary)',
            buttons: 'var(--color-buttons)',
            typography: 'var(--color-typography)',
          },
        },
      },
      plugins: [],
    }
    ```

</Steps>

## Usage

:::note
See [Dynamic theme switching](/design-engineering/tailwind/dynamic-theme-switching) to know more about how to switch between these multiple themes automatically or based on user action.
